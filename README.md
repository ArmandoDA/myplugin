# MyPlugin

Questo plugin per OpenStack invia notifiche via email quando viene attivato o disattivato. Quando attivo invia anche parametri da OpenStack.

## Funzionalità

- Invio email all'attivazione
- Invio email alla disattivazione
- Invio parametri OpenStack via email

## Installazione

1. Clona il repository su Github
2. Installa il plugin con `python setup.py install`
3. Configura le credenziali Gmail nel file `notifier.py`
